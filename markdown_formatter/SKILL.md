# TXT 到 Markdown 格式化技能

此技能专门用于将 txt 文件格式化为结构清晰、易于阅读的 markdown 格式。

## 技能用途

当用户出现以下请求时触发此技能：
- "把这个txt文件转换为markdown"
- "格式化我的文本文件"
- "将这个文件转为markdown格式"
- "改善文档结构和可读性"
- "给文本添加markdown格式"

## 使用流程

### 1. 分析输入文件
- 读取用户提供的 txt 文件内容
- 识别文本结构类型（段落、列表、代码、表格等）
- 检测现有的格式化模式

### 2. 应用格式化规则
使用 `scripts/format_txt_to_md.py` 进行智能格式化：
- 自动识别标题并添加 # 符号
- 将连续的行转换为列表
- 识别代码块并添加 ``` 包装
- 处理表格数据
- 添加适当的空行和分隔符

### 3. 应用模板和样式
根据文档类型选择合适的模板：
- 使用 `assets/templates/` 中的模板
- 参考 `references/formatting_rules.md` 中的规则
- 确保输出符合 markdown 最佳实践

### 4. 输出结果
- 生成格式化后的 markdown 文件
- 保持原始内容的完整性
- 提供格式化摘要说明

## 可重用资源

- 常见模式匹配和处理

详细的格式化规则和最佳实践：

### assets/templates/
各种文档类型的格式化模板：

- 日志文件模板

## 执行步骤

1. 读取并分析输入的 txt 文件
2. 运行格式化脚本处理文本
3. 根据文档类型应用相应模板
4. 验证 markdown 格式正确性
5. 输出格式化后的文件

## 注意事项

- 保持原始内容的语义完整性
- 优先使用最简单的 markdown 语法
- 对于复杂的结构，提供清晰的注释说明
- 确保生成的 markdown 文件可读性和可维护性
