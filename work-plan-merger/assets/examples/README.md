# 工作规划汇总技能示例文件

## 文件说明

本目录包含了工作规划汇总技能的示例文件，用于演示如何使用该技能。

### 文件列表

1. **总纲_年度规划.md** - 总纲规划文件示例
2. **技术研发规划.md** - 技术领域子规划示例
3. **市场拓展策略.md** - 市场领域子规划示例
4. **README.md** - 本说明文件

## 使用方法

### 1. 准备文件
- 将总纲文件放在指定位置
- 将各子规划文件放在同一目录下

### 2. 执行整合命令
```bash
python scripts/merge_plans.py \
    assets/examples/总纲_年度规划.md \
    assets/examples/ \
    整合后的规划汇总.md
```

### 3. 查看结果
- 整合后的规划汇总.md将包含总纲结构
- 各子规划内容会智能匹配到相应章节
- 保持原有的markdown格式和层次

## 预期效果

整合后的文档将按以下结构组织：
```
# 2024年度公司工作规划总纲

## 二、主要工作领域

### 2.1 技术发展规划
--- 子规划内容 ---
### 📋 技术发展规划详细方案
  [技术研发规划的具体内容]

### 2.2 市场拓展策略
--- 子规划内容 ---
### 📋 市场拓展策略执行方案
  [市场拓展策略的具体内容]
```

## 自定义配置

可以通过修改 `references/matching_rules.md` 文件来调整：
- 匹配算法权重
- 关键词映射规则
- 输出格式设置
- 内容分隔符样式
